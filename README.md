# 🛡️ Premium Multiprotocol VPN Infrastructure

Современная, высокоустойчивая к блокировкам VPN-инфраструктура, спроектированная для обхода глубокого анализа пакетов (DPI/ТСПУ) в РФ.

## 🌟 Возможности и архитектура

*   **Параллельный Мост (Relay-Bridge):** Трафик пользователей из РФ пересылается через резидентный/облачный релей в РФ (`sub.synthosai.ru`), а затем распределяется по трем независимым параллельным каналам (VLESS-Reality-Vision, gRPC и xHTTP) до целевого сервера в США. Это исключает проблему «бутылочного горлышка» единственного TCP-соединения и защищает от блокировок по портам.
*   **Умная Маршрутизация (Split-Routing):** Все российские сервисы (Госуслуги, банки, Яндекс, ВК) и ресурсы зоны `.ru` работают напрямую через провайдера без участия VPN. Зарубежные ресурсы (YouTube, Instagram, Google) автоматически маршрутизируются через туннель.
*   **Локальное обслуживание гео-баз:** Базы маршрутизации `geoip.dat` и `geosite.dat` раздаются непосредственно с сервера подписок для обхода блокировок GitHub со стороны ТСПУ.
*   **Оптимизация сокетов:** Агрессивный таймаут простоя (`connIdle` = 90с) и регулярный TCP Keep-Alive (`15с`) на уровне ядра предотвращают накопление «зомби»-соединений от мобильных клиентов.

## 🛠️ Содержимое репозитория

*   `/panel` — веб-панель администратора на FastAPI для ведения базы данных пользователей и генерации динамических подписок.
*   `/bot` — Telegram-бот для автоматической продажи подписок и выдачи конфигураций.
*   `/scripts` — скрипты резервного копирования и синхронизации конфигураций с релеем.
