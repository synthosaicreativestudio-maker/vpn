# 🛡️ Premium Multiprotocol VPN

Современный, устойчивый к блокировкам VPN на март 2026 года. 
Основан на **Sing-Box** и **Xray-core** с поддержкой 4-х разных протоколов для обхода DPI.

## 🌟 Как это работает

Проект использует **Умную Маршрутизацию (Split-Routing)**:
1. Системы, сайты РФ (Госуслуги, Сбербанк, Яндекс) и приложение RIES (Этажи) работают **напрямую** — быстро и без блокировок.
2. Весь остальной мир (YouTube, Instagram, Google) — через **VPN**.
3. Рекламные сети — **блокируются**.

## 🚀 Как начать работу (Клиент)

Мы рекомендуем использовать клиент **Hiddify** на iOS/Android/Windows/Mac.

В проекте сейчас есть только рабочие и реально собранные файлы. Используйте их по назначению:

1. `configs_2026_03_20/raw_sub.txt` или `configs_2026_03_20/sub.txt` — **основная подписка**. Содержит `VLESS tcp`, `VLESS gRPC` и `Hysteria2`.
2. `configs_2026_03_20/hiddify_ALL_IN_ONE.json` — **профиль для Hiddify / sing-box**.
3. `_backup_configs/client_config_global.json` — **глобальный Xray-профиль**.
4. `_backup_configs/client_config_smart_routing_ru.json` — **профиль со smart-routing**.
5. `configs_2026_03_20/qr_amnezia_vless.png` — **QR для прямого VLESS**.

### 🛡️ Резервы
`AmneziaWG / WireGuard` и старые схемы `XHTTP / Shadow-TLS` остаются в архиве проекта. Они не являются частью текущего активного пакета доставки.
