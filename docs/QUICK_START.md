## 🛠️ Управление сервисами

### Сервер подписок (37.1.212.51)

```bash
ssh root@37.1.212.51

# Перезапуск панели
systemctl restart vpn-panel vpn-bot

# Проверка статуса
systemctl is-active vpn-panel vpn-bot caddy vpn-relay-tunnel

# Проверка здоровья
curl http://127.0.0.1:8085/health
```

### VPN-сервер (38.180.81.181)

```bash
ssh root@38.180.81.181

# Перезапуск Xray
systemctl restart xray
# ⚠️ После рестарта Xray — обязательно перезапустить панель!
ssh root@37.1.212.51 "systemctl restart vpn-panel"

# Проверка статуса
systemctl is-active xray caddy vpn-relay-tunnel
```

### РФ-Релей (185.4.67.223)

```bash
# Доступ только через jump host (VPN-сервер или панельный)
ssh root@38.180.81.181
ssh -i /root/.ssh/id_ed25519 ubuntu@185.4.67.223

# Перезапуск Xray на релее
sudo systemctl restart xray

# После рестарта — перезапустить SSH-туннели
exit  # вернуться на VPN-сервер
systemctl restart vpn-relay-tunnel
ssh root@37.1.212.51 "systemctl restart vpn-relay-tunnel vpn-panel"
```

## 📦 Регламент деплоя и управления сборками

Все три сервера используют систему версионирования. Текущая стабильная версия после зачистки CDN — **`v1.0.0`**.

### Процедура выкатки изменений (на примере Xray на VPN)

Если нужно обновить конфигурацию или версию Xray:
1. Подключитесь к `38.180.81.181`.
2. Заархивируйте текущую рабочую конфигурацию в архив версий:
   ```bash
   cp /etc/xray/config.json /etc/xray/builds/config.v1.0.1.json
   ```
3. Отредактируйте рабочий файл `/etc/xray/config.json`.
4. Запустите валидатор: `xray test -config /etc/xray/config.json`.
5. Перезапустите службу: `systemctl restart xray`.
6. Проведите ручное тестирование соединений.
7. **Фиксация сборки:** Если всё работает отлично, перезапишите стабильную копию:
   ```bash
   cp /etc/xray/config.json /etc/xray/config.json.stable
   ```
   *Если вы этого не сделаете, ежеминутный скрипт автоотката посчитает любое зависание новой конфигурации поводом вернуть старую версию `v1.0.0`.*

### Отключение автоотката при технических работах

Если вы собираетесь долго отлаживать конфигурацию, временно отключите проверку в планировщике, чтобы скрипт не откатывал ваши правки:
```bash
# Для отключения (комментирование строки в crontab)
crontab -e
# Поставьте символ # перед строкой xray-failover.sh / panel-failover.sh
```

Не забудьте раскомментировать строку после завершения технических работ!
