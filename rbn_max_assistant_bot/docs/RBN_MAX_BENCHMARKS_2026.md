# RBN MAX Benchmarks 2026

Этот файл подготовлен для RAG в MAX-боте. Он содержит короткие, чистые и структурированные benchmark-правила для первичного анализа коммерческого объекта.

## Главная логика

- Сначала определить сегмент объекта.
- Потом сверить объект с ядром спроса по площади, локации, технике и цене.
- Затем определить: объект в ядре, рядом с ядром или вне ядра.
- После этого дать первичную оценку ликвидности, диапазона цены и окупаемости.

## Что использовать как источники

- `docs/📊 Матрица ядра спроса в коммерческой недвижимости (РФ) основной.xlsx`: Основной эталон. Использовать как master-source по ядру спроса, диапазонам ставок, типам арендаторов и техминимуму
- `docs/6.Комиссии по сделкам_20-04-2026.xlsx`: Факт потока сделок. Использовать для веса сегмента, частоты сделок, подтверждения чистых сегментов
- `docs/66.Все комиссии по сделкам_22-04-2026.xlsx`: KPI по партнерам. Не использовать как benchmark объекта: это файл по суммам комиссий, а не по характеристикам объектов

## Главные выводы по данным РБН

- Для объектной оценки основой являются матрица ядра спроса и факт потока сделок.
- Файл `66.Все комиссии по сделкам_22-04-2026.xlsx` не использовать как benchmark объекта: это отчет по комиссиям партнеров.
- Офисы и торговые помещения являются самыми чистыми и пригодными сегментами для benchmark-сравнения.
- По складам данные использовать осторожно: часть площадей загрязнена.
- ПСН без типологии не считать чистым сегментом.

## Калибровка по факту РБН

- **Офисное помещение**: ядро `10-39 м2 (52%)`, рынок `863-1117 руб/м2`, медиана `35000 руб/мес`, ликвидность `8`. массовый сегмент, малые офисы = ядро.
- **Торговое помещение**: ядро `30-69 м2 (47%)`, рынок `1000-1480 руб/м2`, медиана `70000 руб/мес`, ликвидность `8`. ядро рынка, актив дороже офиса примерно в 3 раза.
- **База/склад**: ядро `50-200 м2 (оценка)`, рынок `650-900 руб/м2`, медиана `28670 руб/мес`, ликвидность `7`. выборка мала, использовать осторожно.
- **ПСН аренда**: ядро `30-59 м2`, рынок `1084-1275 руб/м2`, медиана `37400-69500 руб/мес`, ликвидность `0`. подтверждает экспертную матрицу.

## Правила первичного scoring

- **Типология** (25%): Полное совпадение с чистым сегментом. ПСН без типологии = штраф; чистый сегмент = плюс.
- **Площадь vs ядро** (20%): Попадание в ядро площади. В ядре = 100%; рядом с ядром = 70%; вне ядра = 30%.
- **Цена/ставка vs рынок** (20%): Отклонение от benchmark руб/м2. <=5% = отлично; 5-15% = норм; >15% = риск.
- **Локация** (15%): Совпадение с ядровой локацией. 1 линия, жилой массив, центр трафика, промзона по сегменту.
- **Техпараметры** (10%): Соответствие техминимуму. Мощность, высота, витрина, вытяжка, парковка.
- **Глубина спроса** (10%): Сделок в месяц и срок экспозиции. Чем больше поток и меньше экспозиция, тем выше ликвидность.

Итоговая интерпретация:

- `80-100`: объект в ядре.
- `60-79`: рядом с ядром.
- `40-59`: рабочий, но с рисками.
- `<40`: низкая ликвидность или заметное расхождение с рынком.

## Методы рентабельности и окупаемости

- **Gross Yield**: Годовой арендный поток / цена покупки. Применение: Быстрая первичная оценка аренды или ГАБ.
- **NOI Yield / Cap Rate**: (Аренда - вакансия - OPEX) / цена покупки. Применение: Базовая инвестиционная оценка по мировым практикам.
- **Payback**: Цена покупки / годовой NOI. Применение: Простая окупаемость.
- **Price-to-Rent**: Цена объекта / годовая аренда. Применение: Быстрое сравнение объектов внутри сегмента.
- **Stress Scenario**: Снижение аренды, рост вакантности, рост расходов. Применение: Проверка устойчивости сделки.
- **3 Scenarios**: Быстрая / Рыночная / Оптимистичная. Применение: Для первичного решения собственника и брокера.
