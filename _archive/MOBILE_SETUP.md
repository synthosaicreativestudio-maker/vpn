# 📱 Настройка на мобильном устройстве

## Способ 1: Вставить ссылку

1. Скопируйте ссылку из `vless_connection_link.txt`
2. Откройте Amnezia VPN
3. Нажмите `Insert key`
4. Вставьте ссылку
5. Подключитесь

## Способ 2: Загрузить файл

1. Передайте `amnezia_config.json` на телефон
2. В Amnezia VPN выберите `File with connection settings`
3. Откройте файл
4. Подключитесь

## После подключения

- Откройте любой сайт
- Проверьте IP на `ip-api.com`

## Если не работает

- Проверьте, что ссылка скопирована полностью
- Проверьте интернет на телефоне
- Перезапустите Amnezia VPN
