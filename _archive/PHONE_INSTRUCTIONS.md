# 📱 Инструкция для подключения на телефоне

## Что нужно сделать

### Вариант 1: Вставить ссылку
1. Откройте `vless_connection_link.txt`
2. Скопируйте строку `vless://...`
3. В Amnezia VPN нажмите `Insert key`
4. Вставьте ссылку
5. Подключитесь

### Вариант 2: Загрузить файл
1. Передайте `amnezia_config.json` на телефон
2. В Amnezia VPN выберите `File with connection settings`
3. Откройте файл
4. Подключитесь

## Ссылка

```
vless://eb4a1cf2-4235-4b0a-83b2-0e5a298389ed@37.1.212.51:443?type=tcp&security=reality&sni=www.microsoft.com&pbk=n5E8KcFHjef-ZC2mKjzkVldLJiLrsjfpE1Z-XmLfxH4&fp=chrome&flow=xtls-rprx-vision#VLESS-Reality
```

## Проверка

- Откройте браузер
- Перейдите на `ip-api.com`
- IP должен быть `37.1.212.51`

## Если не работает

- Проверьте, что ссылка скопирована полностью
- Проверьте интернет на телефоне
- Перезапустите Amnezia VPN
