# 🔗 Простой выбор через ссылки

Одна ссылка `vless://` ведет на один и тот же сервер.

```
vless://eb4a1cf2-4235-4b0a-83b2-0e5a298389ed@37.1.212.51:443?type=tcp&security=reality&sni=www.microsoft.com&pbk=n5E8KcFHjef-ZC2mKjzkVldLJiLrsjfpE1Z-XmLfxH4&fp=chrome&flow=xtls-rprx-vision#VLESS-Reality
```

## Как использовать

- Для простого режима: вставьте ссылку и выберите `Global`
- Для smart-routing: используйте `client_config_smart_routing_ru.json`

## Что не считать активным

- `client_config_clash_meta.yaml`
- любые гибридные сценарии
