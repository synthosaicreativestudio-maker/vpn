# 📲 Инструкция: настройка VPN на телефоне

## Что лучше: ссылка VLESS или JSON-конфиг?

| Способ | Плюсы | Минусы | Рекомендация |
|---|---|---|---|
| **VLESS-ссылка** | Один клик — импорт | Нет split-routing на клиенте (только серверный) | ✅ **Для начала — используй это** |
| **JSON-конфиг** | Split-routing на клиенте + блокировка рекламы | Нужно передать файл на телефон | Для продвинутых |

> 💡 **Рекомендация:** начните с VLESS-ссылки — это проще. Split-routing уже работает на сервере, поэтому RU-сайты пойдут напрямую, а иностранные через VPN в обоих случаях.

## 📱 Happ (iOS)

### Способ 1: По ссылке (рекомендуемый)

1. **Скопируйте** VLESS-ссылку выше
2. Откройте **Happ** на iPhone
3. Нажмите **«+»** (добавить сервер)
4. Выберите **«Из буфера обмена»** или **«Import from Clipboard»**
5. Конфигурация появится в списке серверов
6. Нажмите **«Подключить»** ▶️
7. Разрешите создание VPN-профиля, если попросит

### Способ 2: Через JSON-конфиг (продвинутый)

1. Отправьте файл `happ_config_relay_whitelist.json` на iPhone:
   - Через **AirDrop** (самый быстрый)
   - Через **Telegram** (отправьте себе в «Избранное»)
   - Через **iCloud Drive**
2. Откройте файл на iPhone
3. Выберите **«Открыть в Happ»**
4. Конфигурация импортируется автоматически
5. Нажмите **«Подключить»** ▶️

### Где скачать Happ
- App Store: поищите **«Happ»** или **«Happ VPN»**
- Если не находится — возможно, убрали из RU App Store. Используйте **Hiddify** или **V2Box**

## 📱 Hiddify (Android и iOS)

### Где скачать

| Платформа | Ссылка |
|---|---|
| **Android** | Google Play → «Hiddify» или [github.com/hiddify/hiddify-app](https://github.com/hiddify/hiddify-app/releases) |
| **iOS** | App Store → «Hiddify» |

### Способ 1: По ссылке (рекомендуемый)

1. **Скопируйте** VLESS-ссылку выше
2. Откройте **Hiddify**
3. Нажмите **«+»** → **«Add from Clipboard»** (или «Добавить из буфера»)
4. Появится профиль **VPN-Relay-YandexVM**
5. Нажмите **«Connect»** (большая кнопка подключения)
6. На Android: разрешите VPN-подключение во всплывающем окне

### Способ 2: Через JSON-конфиг (продвинутый)

1. Отправьте файл `hiddify_config_relay_whitelist.json` на телефон:
   - **Android**: через Telegram, Google Drive, USB-кабель
   - **iOS**: через AirDrop, Telegram, iCloud
2. Откройте Hiddify
3. Нажмите **«+»** → **«Add config from file»** (или «Импорт из файла»)
4. Выберите скачанный JSON-файл
5. Нажмите **«Connect»**

## ❓ Если не работает

| Проблема | Решение |
|---|---|
| «Не удалось подключиться» | Убедитесь что есть интернет. Попробуйте переключить Wi-Fi ↔ мобильные данные |
| Медленная скорость | Это нормально для цепочки серверов. Для повседневного RU-интернета VPN можно отключать |
| Сайт не открывается с VPN | Некоторые сайты блокируют VPN. Отключите VPN для этих сайтов |
| «Invalid config» при импорте JSON | Убедитесь что файл не был повреждён при передаче. Попробуйте через AirDrop |
