# 📱 Инструкция для Hiddify / Happ

Эти приложения лучше всего работают с автоматической подпиской.

## 1. Автоматическая подписка (Рекомендуется)
Для **Hiddify**: Берите ссылку `http://37.1.212.51:8085/sub/...` из бота.
Для **Happ (iOS)**: Обязательно берите HTTPS-ссылку (`https://37.1.212.51.sslip.io:8086/sub/happ/...`) из бота. Happ требует защищенного соединения HTTPS и формата Base64!

## 2. Самый стабильный прокси (Shadow-TLS v3)
Если подписка не работает, добавьте этот прокси вручную:
`shadow-tls://ShadowPassword2026@37.1.212.51:443?sni=taxi.yandex.ru&version=3#🔐-ShadowTLS-v3`

## 3. Совет по настройке
В настройках Hiddify выберите режим **«Auto Selection»** (или «URL Test») — так приложение само будет переключаться на самый быстрый протокол (TUIC, VLESS или Reality) в зависимости от типа блокировки.
